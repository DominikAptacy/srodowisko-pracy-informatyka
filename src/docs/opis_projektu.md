# FocusFlow – aplikacja do zarządzania czasem skupienia

## Opis projektu

FocusFlow to aplikacja, której celem jest wspieranie użytkowników w pracy
w trybie głębokiego skupienia (deep work). Aplikacja umożliwia planowanie
sesji pracy, blokowanie rozpraszających stron i aplikacji, monitorowanie
czasu pracy oraz analizę produktywności.

Dodatkowo FocusFlow wspiera regenerację poprzez przerwy zgodne z techniką
Pomodoro oraz proste techniki relaksacyjne.

---

## Cele projektu

- zwiększenie koncentracji użytkownika
- ograniczenie cyfrowych rozpraszaczy
- poprawa organizacji czasu pracy
- analiza i wizualizacja produktywności

---

## Funkcjonalności

1. Planowanie sesji skupienia
2. Harmonogramowanie pracy
3. Blokowanie wybranych stron i aplikacji
4. Powiadomienia o rozpoczęciu i zakończeniu sesji
5. Przerwy zgodne z metodą Pomodoro
6. Statystyki i raporty produktywności
7. Integracja z kalendarzem (Google / Outlook)
8. Techniki relaksacyjne
9. Tryb rywalizacji z innymi użytkownikami

---

## Technologie

- Backend: Node.js (Express) lub Python (FastAPI)
- Frontend: React lub Svelte
- Baza danych: MongoDB lub PostgreSQL
